# Student-Record-Management- 
Introduction
       The Student Record Management System is a simple console-based application developed using the C programming language. It is designed to manage student records efficiently through an easy to use menu .

In this system, the Roll Number is generated automatically, so the user only needs to enter the Student Name and Percentage when adding a new record. The application allows users to add, show, modify, delete, save, and sort student records. It also provides options to delete and modify student records using either the Roll Number or the Student Name.

This project demonstrates the use of structures,functions and file concept in C, making it suitable for learning basic data management and (Create, Read, Update, Delete) operations.

Features
1. Add Student Record
Enter the student's Name and Percentage.
Roll Number is generated automatically.
2. Delete Student Record
Delete a student record by Roll Number.
Delete a student record by Student Name.
3. Show Student Record
Display all student records with Roll Number, Name, and Percentage.
4. Modify Student Record
Modify a student record by Roll Number.
Modify a student record by Student Name.
5. Save Student Record
Save all student records to a file.
6. Delete All Student Records
Remove all student records from the file.
7. Sort Student Records
Sort student records by Name, or Percentage.
